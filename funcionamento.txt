# Funcionamento do Sistema Toca do Lobo
## Visão Geral

Sistema de PDV (Ponto de Venda) e gestão para lanchonetes/bares, com:
- Controle de comandas
- Gestão de produtos e categorias
- Controle de estoque
- Impressão de cupons (cozinha e comanda)
- Integração com dispositivos ESP32 para impressão
- Cardápio virtual

## Estrutura do Projeto
- **backend/**
  - main.py: FastAPI API principal
  - models.py: Modelos SQLAlchemy do banco de dados
  - impressao.py: Lógica de impressão ESC/POS
- **frontend/**
  - templates/: Arquivos HTML (index.html, admin.html, cardapio.html)
  - static/: CSS, JS, assets
- **uploads/: Arquivos enviados (imagens)**

## Banco de Dados
- Padrão: SQLite (caixa_bar.db no diretório raiz)
- Alternativa: MySQL/MariaDB via configuração no .env

### Tabelas Principais
1. categorias: Categorias de produtos
2. produtos: Itens do cardápio (nome, preço, estoque, ean, imagem, etc.)
3. comandas: Pedidos (cliente, mesa, status, total, forma de pagamento)
4. itens_comanda: Itens de cada comanda
5. usuarios: Acesso ao sistema (admin/venda)
6. logs: Histórico de ações
7. fechamentos: Fechamentos de caixa
8. notas_fiscais_processadas: XML de NFe processados
9. fila_impressao: Fila para impressão via ESP32
10. dispositivos_esp32: Dispositivos ESP32 para impressão
11. configuracoes: Parâmetros do sistema

## Formatação de Preços
   - Todos os preços são exibidos em Reais (R$) com vírgula como separador decimal
   - Padrão: `R$ X,XX`
   - Função de formatação no frontend: `valor.toFixed(2).replace('.', ',')` precedido de 'R$ '

## Como Iniciar o Sistema
1. Instalar dependências: `pip install -r requirements.txt`
2. Iniciar backend: `cd backend && python main.py` (ou usar iniciar.bat)
3. Acessar via navegador:
   - PDV: http://localhost:8000/
   - Admin: http://localhost:8000/admin
   - Cardápio: http://localhost:8888/
